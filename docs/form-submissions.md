# Accessing Form Submissions

This guide explains how to view and manage form submissions from the website.

## Form Services Used

| Form | Service | Dashboard URL |
|------|---------|---------------|
| Contact Form | Web3Forms | https://web3forms.com/dashboard |
| Volunteer Form | Web3Forms | https://web3forms.com/dashboard |
| Donation Inquiries | Web3Forms | https://web3forms.com/dashboard |
| Newsletter | Mailchimp | https://mailchimp.com |

## Web3Forms (Contact & Volunteer Forms)

### Viewing Submissions

1. Go to https://web3forms.com
2. Log in with your account credentials
3. Click on "Submissions" in the dashboard
4. View all form submissions with:
   - Sender name and email
   - Message content
   - Timestamp
   - Form type

### Email Notifications

Web3Forms sends submissions directly to your email:
- **Primary**: info@printforachild.org
- **CC**: (add additional recipients in Web3Forms dashboard)

### Exporting Data

1. Go to Submissions
2. Select date range
3. Click "Export CSV"

## Mailchimp (Newsletter)

### Viewing Subscribers

1. Go to https://mailchimp.com
2. Log in with your account
3. Go to Audience → All contacts
4. View all newsletter subscribers

### Sending Newsletters

1. Go to Campaigns → Create Campaign
2. Choose your audience
3. Design your email
4. Send or schedule

### Managing Subscribers

- Unsubscribe requests are handled automatically
- Export subscriber list: Audience → Export Audience

## Troubleshooting

### Not receiving form emails?

1. Check your spam/junk folder
2. Verify email settings in Web3Forms dashboard
3. Confirm the correct email is configured

### Forms not working?

1. Verify the API key is correct in the code
2. Check Web3Forms dashboard for errors
3. Contact development team if issues persist

## Security Notes

- Never share API keys publicly
- Change passwords regularly
- Review submissions for spam periodically
